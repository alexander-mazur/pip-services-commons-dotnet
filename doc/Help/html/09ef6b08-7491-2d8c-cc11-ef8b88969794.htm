ICounters Methods

The ICounters type exposes the following members.

Methods
  NameDescription
BeginTiming
Starts measurement of execution time interval. The method returns ITiming object that provides endTiming() method that shall be called when execution is completed to calculate elapsed time and update the counter.
Increment
Increments counter by specified value. This counter can be used to track various numeric characteristics
IncrementOne
Increments counter by value of 1. This counter is often used to calculate number of client calls or performed transactions.
Last
Records the last reported value. This counter can be used to store performance values reported by clients or current numeric characteristics such as number of values stored in cache.
Stats
Calculates rolling statistics: minimum, maximum, average and updates Statistics counter. This counter can be used to measure various non-functional characteristics, such as amount stored or transmitted data, customer feedback, etc.
Timestamp
Records specified time. This counter can be used to tack timing of key business transactions as reported by clients.
TimestampNow
Records the current time. This counter can be used to track timing of key business transactions.
